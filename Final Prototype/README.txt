This directory corresponds to the final version of this project (i.e. after 2nd iteration), consisting of:

- Final_Prototype.py: The actual program file written in Python script that can be run.

- Final_Prototype.pdf: The printed version of the Python script.

- teleprompter.ico: The icon displayed on the left of the program's title "PROMPT ME OUT!", note that this is invisible to macOS users.

- ValueError1.mp3: The system's sound output message when the last occurrence of a specific word in the text has been entered, in which the sound will get updated per message.
- ValueError2.mp3: The system's sound output message when the input word cannot be found inside the text.
- IndexOutOfBoundError.mp3: The system's sound output message when the last word of the text has been uttered.
- output.mp3: The system's sound output message when a word in the text has been entered, in which the sound will get updated per message.
- UnknownValueError.mp3: The system's sound output message when no input has been detected for more than 8 seconds.

Details regarding environment set-up and installation can be found within the User Guide chapter of the Appendix.

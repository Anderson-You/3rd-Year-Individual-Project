This directory corresponds to the initial version of this project (i.e. before the interview), consisting of:

- Initial_Prototype.py: The actual program file written in Python script that can be run.

- Initial_Prototype.pdf: The printed version of the Python script.

- word.mp3: The system's sound output message when a word in the text has been entered, in which the sound will get updated per message.
- error.mp3: The system's sound output message when the input word cannot be found inside the text.
- exception1.mp3: The system's sound output message when no input has been detected for more than 5 seconds.
- exception2.mp3: The system's sound output message when the last word of the text has been uttered.

Details regarding environment set-up and installation can be found within the User Guide chapter of the Appendix.
